Materi

Pemerograman web adalah proses pembuatan program komputer yang bisa diakses menggunakan browser (aplikasi berbasis web). HTML adalah bahasa standar yang digunakan oleh browser internet untuk membuat halaman dan dokumen pada sebuah web yang dapat diakses dan dibaca seperti artikel. HTML juga digunakan sebagai link antara file dalam situs atau dalam komputer dengan menggunakan localhost atau link yang menghubungkan antar situs dalam dunia internet. CSS adalah bahasa pemerograman yang digunakan untuk mengendalikan beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. Adapun kelebihan HTML5 yaitu mampu menangani error secara lebih baik dibandingkan dengan HTML dan HTML juga menggunakan syntax yang lebih sederhana dibanding HTML sehingga developer bisa membuat struktur halaman website yang kompleks secara lebih mudah.
